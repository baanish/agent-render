agent-render self-hosted API

The optional self-hosted server exposes REST endpoints under /api for storing artifact payloads in SQLite and serving viewer links. Build and run with npm run selfhosted:dev, or use the Docker image described in the repository.

Machine-readable description: OpenAPI at /openapi/selfhosted-artifacts.yaml. See the skills/selfhosted-agent-render/SKILL.md file in the project repository for full usage.
